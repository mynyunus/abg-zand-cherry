# Landing Page Chery - Abg Zand

Landing page statik (HTML/CSS/Vanilla JS) untuk lead WhatsApp dealer kereta Chery.

## Struktur Fail

- `index.html`
- `styles.css`
- `script.js`
- `assets/` (gambar placeholder diperlukan)

## Asset Placeholder Diperlukan

Pastikan fail berikut wujud:

- `/assets/sales.jpg`
- `/assets/hero-car.jpg`
- `/assets/car-1.jpg` hingga `/assets/car-12.jpg`
- `/assets/testi-1.jpg` hingga `/assets/testi-6.jpg`
- `/assets/key-1.jpg` hingga `/assets/key-3.jpg`

## Deploy ke Cloudflare Pages

1. Push projek ke GitHub/GitLab.
2. Di Cloudflare Pages, pilih **Create a project** dan connect repository.
3. Tetapan build:
   - **Framework preset:** `None`
   - **Build command:** `none` (atau kosongkan)
   - **Build output directory:** `/`
4. Deploy.

Tiada build step diperlukan kerana ini laman statik sepenuhnya.
